## How to use this file

The trauma layer — and a file that is mostly about **holding a boundary**. CFT for trauma is
phased, specialist work; the techniques that process trauma (imagery rescripting,
compassionate dialogue with the child self, FBR work) are **clinician-only**
(`advanced-clinician.md`, `safety.md`). In self-help you deliver only stabilization and
psychoeducation, and you signpost. Assumes `model.md` and `core-cmt.md`. Read `safety.md`
before anything. **Safety:** if there's active risk, dissociation you can't ground, or the
user wants to process a memory, stop and follow `safety.md`.

## What CFT targets in trauma

CFT's specific target in trauma is **shame**, not the trauma memory per se: the self-blame
("something about me caused or enabled this"), the felt sense of being damaged, and the
inability to receive compassion that trauma generates. It formulates PTSD as both external
threat (a hypervigilant threat system) and internal shame. This framing alone — *the shame
is a wound, not the truth about you; this is not your fault* — can be de-shaming when offered
gently (`model.md` 2).

## The self-help-safe layer only

What this skill can safely do, and only this:

1. **Grounding and stabilization** — 5-4-3-2-1, feet on the floor, an object in the hand, for
   when the threat system spikes or dissociation looms.
2. **Soothing Rhythm Breathing** (`core-cmt.md` 1) — the foundational down-regulation skill.
3. **Safe / Soothing Place** (`core-cmt.md` 2) — *sensory grounding only*; for trauma
   histories, stay in the sensory steps and do **not** progress to the "the place is glad
   you're here" reciprocity step, which can stir attachment material too soon.
4. **De-shaming psychoeducation** (`model.md`) — the threat/soothing model and "not your
   fault", paced and gentle.
5. Slowly building a **Compassionate Self** (`core-cmt.md` 5) is the preparation a clinician
   would want in place before any rescripting — but build it as capacity, not as a route into
   the memory.

## What is clinician-only here

Move through the ladder *very slowly*; trauma (especially sexual trauma) brings pronounced
shame and strong fears/blocks/resistances to compassion. **Do not**, as self-help: guide the
user into a trauma memory, run imagery rescripting, run chairwork or compassionate dialogue
with the child self, or do FBR exposure. These are CFT-trained-clinician work
(`advanced-clinician.md`). The phased CFT trauma protocol (group resilience → individual
processing → identity consolidation) is named here so the user knows the path exists — to
walk with a therapist.

## If it stalls

- Backdraft or a flashback surfaces → stop the practice, ground, SRB, do not interpret the
  memory; if it doesn't settle or there's risk, follow `safety.md`.
- The user wants to "deal with" the memory now → validate the wish, hold the boundary warmly:
  this is exactly the work that needs a trained clinician and a real therapeutic relationship;
  offer stabilization and signpost.
- Can't access any soothing place → stay with SRB and the **Compassionate Kitbag**
  (`core-cmt.md` 10) as a body anchor; imagery may be too soon.

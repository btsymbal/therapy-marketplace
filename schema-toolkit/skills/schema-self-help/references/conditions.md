## How to use this file

`routing.md` routes by the pattern in the room; this file is for the harder, named presentations — when the user has a diagnosis, when the picture is complex, or when you need to set honest expectations about what self-help can and can't do. For each condition it gives the characteristic schema/mode profile, the technique subset Schema Therapy reaches for first, and where the evidence is genuinely strong versus thin. The recurring message: Schema Therapy's strongest evidence is for personality-disorder presentations *delivered by a clinician over a long course* — this skill offers the self-help-safe slice and a knowledgeable signpost, not a substitute for that treatment. Techniques referenced live in the other files; the model is in `core-model.md`.

## Condition → schema/mode profile → first-line subset

| Condition | Characteristic schemas | Key modes | First-line technique subset |
|---|---|---|---|
| **Borderline PD** | Abandonment, Mistrust/Abuse, Defectiveness/Shame, Emotional Deprivation | Abandoned/Abused Child, Angry/Impulsive Child, Detached Protector, Punitive Parent, (weak) Healthy Adult | Self-reparenting + imagery rescripting + chair work across all modes; flashcards & a transitional object for abandonment spikes |
| **Cluster C PDs** (avoidant, dependent, OCPD) | Emotional Inhibition, Subjugation, Defectiveness, Unrelenting Standards | Detached Protector, Detached Self-Soother, Compliant Surrenderer, Vulnerable Child | Imagery rescripting + chair work (Detached Protector, Demanding/Critical Parent) + historical role play + self-reparenting |
| **Narcissistic PD** | Defectiveness (hidden), Entitlement, Emotional Deprivation | Self-Aggrandizer/Overcompensator, Detached Self-Soother, hidden Abandoned/Abused Child | Internal empathic confrontation (more direct) + imagery for the child beneath the grandiosity + self-reparenting |
| **Chronic / treatment-resistant depression** | Defectiveness, Failure, Unrelenting Standards, Subjugation | Demanding/Critical Parent → Vulnerable Child → Detached Protector | Chair work (Demanding Parent vs. Child/Healthy Adult) + imagery rescripting + schema-level restructuring + behavioral activation aligned to Healthy-Adult values |
| **Complex PTSD / childhood trauma** | Defectiveness/Shame, Vulnerability to Harm, Mistrust/Abuse, Social Isolation | Vulnerable Child, Detached Protector, Punitive Parent | Trauma-focused imagery rescripting + self-reparenting + empathic confrontation of shame — **clinician-led at depth**; self-help = stabilization + identification only |
| **Eating disorders** (when CBT-E insufficient) | Defectiveness/Shame, Emotional Deprivation, Abandonment, Subjugation | Vulnerable Child, Over-Controller, Self-Critical, Detached Avoidant | Imagery rescripting + chair work (Over-Controller) + behavioral pattern-breaking — **alongside CBT-E + medical care, not instead** |
| **Anxiety / OCD** (treatment-resistant) | Vulnerability to Harm, Self-Sacrifice, Approval-Seeking, Social Isolation | Vulnerable (Anxious) Child, Detached Protector, Compliant Surrenderer | Schema psychoeducation + imagery rescripting for the developmental root + empathic confrontation of the mode blocking exposure + behavioral pattern-breaking — usually integrated with CBT |
| **Substance use** | Insufficient Self-Control, plus the schema being self-medicated | Detached Self-Soother, Impulsive/Undisciplined Child | Schema identification + behavioral pattern-breaking (alternatives to the schema-avoidance function) + flashcards for craving + imagery if childhood trauma underlies it |

## Per-condition notes and honesty

- **BPD** — the most thoroughly studied: multiple strong (Tier-1) RCTs, large effects vs. treatment-as-usual, roughly on par with DBT. But the evidence is for *full clinician-delivered ST over ~2-3 years* (often individual + group; group alone underperforms). Self-help here is an adjunct for between-session regulation (flashcards, diary, grounding) and understanding — not the treatment. Signpost clearly.
- **Cluster C** — strong Tier-1 evidence; ST superior to treatment-as-usual at follow-up. A validated brief protocol exists (~18 sessions) but is clinician-delivered. Good condition for the identification + cognitive + behavioral self-help layer between sessions.
- **NPD** — theoretically well-developed, **no standalone RCT** (the one trial's NPD subgroup was tiny). Empathic confrontation is more direct than in BPD; the work is reaching the abandoned/abused child *under* the grandiosity. Treat as promising-but-unproven.
- **Chronic depression** — defensible claim is *non-inferiority* to alternatives, not superiority; high residual symptoms even after treatment. Watch the Demanding/Critical Parent → Vulnerable Child → Detached Protector sequence specifically. Pair experiential work with Healthy-Adult-values behavioral activation.
- **Complex PTSD / trauma** — the biggest gap between theoretical fit and evidence; **no completed standalone RCT**, rated the field's top research priority. Trauma-focused imagery rescripting is firmly clinician territory. Self-help = grounding, stabilization, schema/mode identification; **signpost a trauma-competent clinician**.
- **Eating disorders** — schema relevance is established; **RCT evidence pending**. Position ST as second-line/augmentation when CBT-E or DBT prove insufficient, always with medical monitoring. Do not run a weight/shape-control frame.
- **Anxiety / OCD** — promising pilots only, **no standalone ST RCT**; usually integrated with CBT for treatment-resistant cases. Schema psychoeducation helps explain why standard exposure felt intolerable.
- **Substance use** — schema links established (Insufficient Self-Control most associated), but **no dedicated ST-only RCT**; evidence is from a record review, not a trial. Useful for the schema *driving* the use; combine with appropriate addiction care.
- **Forensic / couples** — specialized (forensic mode extensions; "schema chemistry" for couples). Evidence is very limited (forensic: "very low" certainty) to conceptual-only (couples). Out of scope for this skill beyond awareness; refer to specialist services.

## Out of scope / specialist referral

Some presentations need more than self-help, and saying so plainly is part of the work:

- **Eating disorders** — needs specialist CBT-E/specialist ED service + medical monitoring. Signpost; deliver only stabilization/identification.
- **Active substance dependence needing detox**, **untreated psychosis/mania**, **acute suicide risk**, **recent interpersonal violence** — these are also hard contraindications for experiential work (`safety.md`); route to appropriate acute/specialist care.
- **Trauma-focused experiential work on severe childhood trauma**, **BPD-level rapid mode switching**, **forensic risk** — a qualified schema therapist (and often a structured program) is the real treatment. Offer the safe layer (psychoeducation, identification, flashcards, diary, grounding, Healthy-Adult building) and actively encourage professional ST. Use [findahelpline.com](https://findahelpline.com) for crisis, and suggest the ISST therapist directory (isst-online.com) as a way to find a certified schema therapist when they're ready.

The honest frame throughout: this skill helps a person understand their schemas and modes, regulate between sessions, and rehearse Healthy-Adult responses. For the conditions above, that is a valuable companion to professional Schema Therapy — not a replacement for it.

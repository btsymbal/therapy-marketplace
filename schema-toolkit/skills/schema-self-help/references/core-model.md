## How to use this file

This is the shared vocabulary every technique file assumes — the schema-therapy equivalent of knowing the alphabet before you read. You rarely teach it wholesale; you use it as a lens to locate what's firing, and you pull the relevant piece in as psychoeducation when it helps the user make sense of their experience. The glossary tables (schemas, modes) also give the **canonical English tags** the log uses (`logging.md`), so patterns stay greppable across sessions and languages. Deliver any teaching of this per `walkthrough-protocol.md`: a sentence or two, tied to their concrete example, not a lecture.

The central idea: painful or unmet early experiences lay down **Early Maladaptive Schemas** — broad, self-perpetuating beliefs about oneself and others — and these drive persistent emotional pain, relationship trouble, and self-defeating behavior. The work names and validates the pattern, shifts its emotional charge, and builds new responses from a **Healthy Adult** stance.

## The four constructs in one breath

1. **Early Maladaptive Schemas (EMSs)** — broad themes about self and others, formed in childhood, dysfunctional now. Not just a belief: a memory-cognition-emotion-body complex that was once the best available adaptation and is now a prison.
2. **Core emotional needs** — five universal childhood needs whose frustration produces schemas.
3. **Coping styles** — the three habitual ways people respond when a schema is triggered (surrender, avoidance, overcompensation).
4. **Modes** — the moment-to-moment emotional states a person flips into when a schema fires. This is usually what you're working with in the room.

## The 5 core emotional needs

When frustrated in childhood, these produce the schemas. Naming the unmet need is itself validating and orients the whole work:
1. Secure attachment (safety, stability, nurturance, acceptance)
2. Autonomy, competence, and a sense of identity
3. Freedom to express valid needs and emotions
4. Spontaneity and play
5. Realistic limits and self-control

(Honesty note: the causal chain from unmet need → schema is foundational to the model but only lightly tested empirically. Use it as a meaning-making frame, not a proven mechanism.)

## The 3 coping styles

When a schema is triggered, people respond through one habitual pattern. Map roughly to freeze / flight / fight. Log tag in brackets.
- **Surrender** `[surrender]` — accept the schema as true and act accordingly (e.g. stay with someone who confirms Mistrust).
- **Avoidance** `[avoidance]` — avoid activating situations: withdrawal, numbing, distraction, substances (the Detached modes live here).
- **Overcompensation** `[overcompensation]` — behave as the opposite of the schema (grandiosity over Defectiveness, control over Vulnerability).

The same schema shows up very differently depending on coping style — spotting the style tells you which mode you'll meet.

## The modes (+ log tags)

A mode is the momentary state the person is *in* when a schema fires. In session you usually work the mode, not the abstract schema. Name it in their language (encourage personally meaningful labels — "the little one", "the cold wall", "the drill sergeant"); log the canonical tag.

| Category | Mode | Log tag |
|---|---|---|
| **Child modes** | Vulnerable Child (lonely, abandoned/abused, humiliated, anxious) | `vulnerable-child` |
| | Angry Child (explosive anger at unmet needs) | `angry-child` |
| | Impulsive/Undisciplined Child (acts on whims, can't tolerate frustration) | `impulsive-child` |
| | Happy Child (needs met, content, playful) | `happy-child` |
| **Dysfunctional parent modes** | Punitive Parent (self-hatred, shame, "you deserve punishment") | `punitive-parent` |
| | Demanding/Critical Parent (relentless standards, "not good enough") | `demanding-parent` |
| **Dysfunctional coping modes** | Compliant Surrenderer (gives in, people-pleases, self-erases) | `compliant-surrenderer` |
| | Detached Protector (numbs, withdraws, cuts off feeling) | `detached-protector` |
| | Detached Self-Soother (soothes via distraction/substances/compulsion) | `detached-self-soother` |
| | Overcompensator / Self-Aggrandizer (grandiose, controlling, attacking) | `overcompensator` |
| **Healthy modes** | Healthy Adult (the goal — see below) | `healthy-adult` |
| | Happy Child | `happy-child` |

(Forensic extensions — Angry Protector, Conning/Manipulative, Predator — exist for specialized settings; not relevant to general self-help.)

A typical sequence is worth mapping explicitly (this is the case conceptualization, see `assessment.md`): e.g. *trigger → Vulnerable Child activated → Detached Protector shuts it down → later Angry Child erupts*. Identify the parent-child dyads too (which critic mode triggers which child).

## The Healthy Adult

The overarching goal of all the work is to **strengthen the Healthy Adult mode** — the part that nurtures the Vulnerable Child, sets limits on the Impulsive Child, and stands up to the Punitive/Demanding Parent. It has seven dimensions (Edwards 2022): meta-awareness, reality orientation, agency, emotional connection, reciprocity, coherent identity, and compassion. Early on you (and a therapist) lend Healthy-Adult functioning by modeling it; the marker of progress is the user accessing their own. See `healthy-adult.md`.

## The mechanisms of change and the three phases

**Four mechanisms** (Kellogg & Young): (1) limited reparenting / self-reparenting; (2) experiential imagery and dialogue work (imagery rescripting, chair work, letter writing); (3) cognitive restructuring and education; (4) behavioral pattern-breaking.

**Critical timing principle:** cognitive and behavioral change land *after* the dysfunctional coping modes are softened and the emotional charge is worked experientially — not at the start while a Detached Protector is fully up. This is why routing often goes: name/validate the coping mode → access the Vulnerable Child experientially → then cognitive and behavioral consolidation.

**Three phases** (a useful arc, not a rigid schedule for self-help):
- **Phase 1 — Bonding & exploration:** psychoeducation, identifying schemas and modes, building the mode map, early experiential access. (`assessment.md`)
- **Phase 2 — Change:** intensive experiential work (`experiential.md`, `letter-writing.md`), then schema-level cognitive restructuring (`cognitive.md`) once coping modes are dismantled.
- **Phase 3 — Consolidation & autonomy:** behavioral pattern-breaking (`behavioral.md`), Healthy-Adult strengthening (`healthy-adult.md`), and the user increasingly doing it themselves.

## Honesty caveats

- **Mode counts proliferate.** The original model had ~10-11 modes; inventories assess 14; some sources claim 40+ or 80+. The extended counts lack a clear empirical basis. Stick to the core set above; don't over-taxonomize a user into invented modes.
- **The unmet-needs → schema causal chain is under-tested** (see the needs section). Use it as a meaning-making frame.
- **Schemas are lifelong patterns.** Self-help softens charge, builds awareness, and rehearses Healthy-Adult responses; it doesn't dissolve a decades-old schema in a session. Set expectations accordingly (`safety.md`).

## How to route

ACT routes by **process, not diagnosis**. The user won't name a hexaflex process, they'll bring a feeling, a stuck thought, an urge, or a sense of drift ("I can't stop thinking about it", "I just want this feeling gone", "what's even the point", "I know what I should do but I can't move"). Your job: from what they say plus the concrete moment, identify which **inflexibility move** is live — fusion, avoidance, lost in time, self-as-story, unclear values, inaction — and reach for that process's **first-line technique**, then open its file. Pick one technique to start, not a menu. Re-route freely if their answers point to a different process.

Always do the safety scan (`safety.md`) and get the experience + a 0-100 intensity (or believability) + a concrete moment first (`SKILL.md` flow steps 1-2). If they're acutely flooded, ground first (`present-moment.md` dropping anchor) before any other process.

## Front door: what the user brings → hexaflex process

| What they bring | Live process (target) | File |
|---|---|---|
| A sticky / fused / intrusive thought they keep buying; rumination; self-criticism; "I can't stop thinking that…" | **Defusion** (get distance from the thought) | `defusion.md` |
| Fighting, suppressing, or running from a painful feeling; "I just want it gone"; grief, anxiety, shame they're bracing against | **Acceptance / willingness** (make room for the feeling) | `acceptance.md` |
| Lost in past or future, planning/rehearsing/replaying, dissociated, "not here", acutely overwhelmed and can't think | **Present-moment contact** (come back, ground) | `present-moment.md` |
| Fused to a self-story / identity: "I am broken / a failure / a burden / unlovable"; over-identified with a label | **Self-as-context** (you are not your story) | `self-as-context.md` |
| Drifting, empty, "what's the point", don't know what matters, going through the motions, values-disconnected | **Values** (find the compass heading) | `values.md` |
| Knows what matters but is stuck, avoidant, inert; can't start; commitment keeps breaking | **Committed action** (one workable step toward it) | `committed-action.md` |
| White-knuckling a craving / urge (substance, food, scroll, self-harm impulse) | **Acceptance** (urge surfing) + `substance-use.md` | `acceptance.md`, `substance-use.md` |
| Pain dragging mood down, struggle with the body, life narrowing around it | **Acceptance + values-based action** | `pain.md` |
| Intrusive thoughts + compulsions, "I have to check / neutralize or something bad happens" | OCD (clinician-guided): defusion psychoeducation only | `ocd.md` (read `safety.md`) |
| Trauma memories, flashbacks, dissociation, hypervigilance | PTSD / trauma (clinician-guided): grounding only | `trauma.md` (read `safety.md`) |
| Generically "stuck" / off with no clear flavour yet | Needs triage | ask one locating question, then route (usually defusion, avoidance, or values-drift) |

If two rows fit, ask one clarifying question to pick, or start with the one tied to the most acute experience. A common pair: a hot thought (defusion) sitting on top of a feeling being fought (acceptance) — start where the heat is.

## Process → first-line technique → file

The subset to reach for **first** in each process. Each technique lives in the listed file.

| Process | First-line technique(s) | File |
|---|---|---|
| Defusion | Labelling ("I'm having the thought that…"), leaves on a stream, milk-milk-milk, naming the story | `defusion.md` |
| Acceptance / willingness | Expansion (making room), dropping the rope, the struggle switch, urge surfing | `acceptance.md` |
| Present-moment contact | Dropping anchor (ACE), five senses grounding, mindful breathing | `present-moment.md` |
| Self-as-context | The observer exercise, sky and weather, chessboard, "I am noticing X" | `self-as-context.md` |
| Values | Values card sort, bull's-eye / life compass, the 80th-birthday / eulogy exercise | `values.md` |
| Committed action | SMART-ACT step, willingness-to-act rating, FEAR/DARE, HARD barriers | `committed-action.md` |

The processes interlock: defusion and acceptance (**Open**) usually come before values and committed action (**Engaged**) land, with present-moment and self-as-context (**Aware**) as the vantage point throughout. If values work activates avoidance, back up to Open work first.

## Condition → first-line process map

When the user does present with a recognizable problem, this maps it to the process(es) ACT reaches for first (distilled from the source's condition map). Still deliver by process, via the files above.

| Presentation | First-line ACT process(es) | Notes / file |
|---|---|---|
| Depression / low mood | Values + committed action; defusion | Values card sort + values-based behavioural activation; defusion for the "what's the point" story |
| Anxiety / chronic worry | Defusion; acceptance; present-moment | Labelling, leaves on a stream, expansion, mindful breathing |
| Panic | Acceptance; defusion | Expansion, dropping the rope, struggle switch — make room for the body sensations rather than fighting them |
| Social anxiety | Defusion; values; committed action | Labelling, passengers on the bus, then values-linked graded action |
| Chronic pain | Acceptance / willingness; values-based action | `pain.md` — willingness + quota-contingent pacing; set expectations per `safety.md` |
| Substance use / craving | Acceptance; defusion; values | `substance-use.md` — urge surfing, defusion from craving-thoughts, Choice Point / BOLD |
| Grief / unavoidable loss | Acceptance; values | Expansion / making room; values to re-engage life alongside the loss |
| OCD | Defusion (clinician-guided) | `ocd.md` — psychoeducation + labelling only; ERP is clinician-led; see `safety.md` |
| PTSD / trauma | Acceptance; defusion; values (clinician-guided) | `trauma.md` — dropping anchor / grounding only as self-help; see `safety.md` |
| Specific phobia | Not a supported ACT indication | Signpost CBT-based exposure (one-session treatment); don't improvise an ACT protocol |
| Eating disorder | Out of scope here | `safety.md` — signpost specialist assessment; do not run the urge frame on restriction/purging |

## Multi-process and re-routing

- Most sessions sit mainly in one process; many are layered (a fused thought *and* a fought feeling; values-drift *and* inaction). Work one at a time, the most acute first. Name the others and offer to come back.
- If a walk-through reveals the real live process is different from where you started (you opened on defusion, but it's actually avoidance of the feeling underneath), say so plainly and switch files. Don't finish the wrong technique for tidiness.
- When distress is too high to do any experiential work, route to **present-moment** down-regulation first (`present-moment.md` dropping anchor / 5-4-3-2-1), then back.
- When the problem is genuinely real and solvable (a concrete task, a real decision), the ACT answer is **committed action** — a workable values-linked step (`committed-action.md`) — not more acceptance of a fixable situation.

## The triflex as a backstop

If you can't cleanly tell which of two adjacent processes is live (defusion vs acceptance, present-moment vs self-as-context), it matters less than the literature once implied — they overlap. Fall back to the triflex arc and just keep moving through it: **Open** (drop the struggle with thoughts and feelings) → **Aware** (notice from the observer's vantage, here and now) → **Engaged** (move toward what matters). Working the arc beats agonizing over the label. See `hexaflex.md`.

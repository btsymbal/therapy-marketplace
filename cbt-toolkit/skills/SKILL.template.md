# Example Skill

Replace this with the skill's instructions.

To create a real skill, make a folder `cbt-toolkit/skills/<skill-name>/SKILL.md`
and fill in the frontmatter above (`name` + `description` are required) plus the
body with the actual guidance. The skill becomes invocable as
`/cbt-toolkit:<skill-name>` once the plugin is installed.

This template file is not itself a working skill — it's here as a starting point.

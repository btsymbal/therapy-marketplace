## How to use this file

The specialized layer for **specific phobias** (animals, heights, flying, enclosed spaces, blood/injury). It assumes the transdiagnostic core in `core-toolkit.md`, especially **graded exposure** (technique 7), and only adds the phobia-specific shape: in-vivo exposure as first-line, systematic desensitization as the imaginal fallback. Deliver every step per `walkthrough-protocol.md`: **one step per turn, end on a question, wait.**

---

## 1 · In-vivo graded exposure for phobias

**Reach for it when:** a circumscribed fear of a specific object or situation kept alive by avoidance. This is first-line and the most effective approach for specific phobias (largest effect sizes). It is the general graded-exposure protocol (`core-toolkit.md` technique 7) run **in-vivo**, with the real feared thing, not imagined; the layer below is the phobia-specific tuning.

**Walk it through:**
1. Set the concrete goal at the top of the ladder (e.g. "stand on a high balcony", "hold the spider"). Ask: "What's the thing you want to be able to do but currently can't?"
2. Build the fear ladder, 10-15 feared/avoided situations from mild to that goal, SUDS 0-100 each, ordered lowest → highest. Ask: "What's a version of this that's only mildly scary, a 30? And what's the worst, the 100?" (fill the middle from there).
3. Break any big jump into smaller sub-steps so no rung is a leap. Ask: "Between those two, what's a middle rung?"
4. Start at the bottom, around SUDS 30. Enter the real situation and **stay** long enough for anxiety to drop noticeably (often 20-30 min, e.g. on the balcony), no escape, no distraction, no safety behaviors. Ask: "Where's the SUDS now, start, and now that you've stayed a while?"
5. Repeat the same rung, frequently and prolonged, until it reliably barely registers, then move up. Ask: "Has this rung gone flat enough to step up?"

**If it stalls:** rung too steep → smaller sub-steps, or more frequency/duration. Anxiety won't drop → strip hidden safety behaviors and distraction; reframe to inhibitory learning (the point is violating the feared prediction, not waiting out anxiety). Can't approach in-vivo at all yet → **systematic desensitization** (2) or imaginal/VR first, then transfer to in-vivo. Consider therapist-assisted, modeling, or one-session massed exposure.

## 2 · Systematic desensitization

**Reach for it when:** in-vivo exposure is too steep to begin, or the phobia (animals, heights, flying) suits a slower imaginal approach. It pairs relaxation with a graded **imaginal** hierarchy (reciprocal inhibition, Wolpe). Note it is generally **less potent** than in-vivo (1) and poorly suited to broad anxiety; treat it as a runway to in-vivo, not a destination.

**Walk it through:**
1. Phase 1, build the relaxation skill first: Jacobson's progressive muscle relaxation (the PMR walk-through is in `core-toolkit.md` technique 9). Ask: "Can you get to a reliably relaxed state on cue? Let's check it before we use it."
2. Phase 2, build a fear hierarchy of 10-15 situations, least → most feared. Ask: "What's the least-scary version of this, and what's the most?" (fill the gradient).
3. Phase 3, while relaxed, visualize the **least** feared item vividly. Ask: "Picture it in detail, what's the SUDS as you hold the image?"
4. If anxiety rises, drop the image and return to relaxation before resuming. Ask: "Did anxiety spike? Let's reset to calm first, then go back in."
5. Move up only when the current item no longer generates anxiety while relaxed. Ask: "Does this image sit flat now? Ready for the next rung?"

**If it stalls:** progress is slow or gains don't transfer to real life → switch to **in-vivo graded exposure** (1), which is generally more effective. Imagery is hard to hold → add VR exposure. Relaxation itself raises anxiety → see the relaxation caveats in `core-toolkit.md` technique 9.
